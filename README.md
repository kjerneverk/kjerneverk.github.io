# Kjerneverk Documentation Site

This repository contains the documentation website for the Kjerneverk project suite, hosted at [https://kjerneverk.github.io](https://kjerneverk.github.io).

## Overview

Kjerneverk (Norwegian for "core work") is a collection of interconnected tools designed to enhance AI-assisted development through:

- **RiotPlan** - Framework for long-lived, stateful AI workflows
- **RiotPrompt** - Structured prompt building library and CLI
- **RiotDoc** - Template-driven document creation with AI assistance

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions.

## Technology Stack

- React 19
- TypeScript
- Vite
- React Router
- React Markdown
